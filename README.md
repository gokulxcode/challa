# E-commerce Student Dashboard & Forecasting System

This repository contains a full-stack demo project:
- Backend: FastAPI + SQLAlchemy + SQLite
- Frontend: React + Vite
- AI: forecasting (statsmodels) + recommendation (sentence-transformers)
- Image upload for products (admin)
- JWT auth, sample data, tests, Dockerfiles, and CI workflow.

See `backend/README-backend.md` and `frontend/README-frontend.md` for details.
